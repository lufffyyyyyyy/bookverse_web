# BookVerse - Live Demo Script

**Estimated Duration:** 3-5 Minutes

## 1. Home Page & Discovery
1.  **Start at `index.html` (Logged out state).**
2.  **Highlight the Design:** Point out the clean layout, the hero section, and the book grid.
3.  **Demonstrate Randomness:** Refresh the page once to show the book order changing.
4.  **Search:** Type "Harry" into the search bar. Show how the grid filters instantly.
5.  **Category Filter:** Hover over "Categories" in the nav. Click "Fantasy". Show the filtered results.

## 2. Registration & Login
1.  **Click "Login"** in the nav.
2.  **Show "Create an account"** link (briefly go to Register page, then back).
3.  **Log in** with a demo user (e.g., `test@example.com` / `password`).
4.  **Point out the Nav Change:** "Login" changes to "My Profile" and "Logout".

## 3. Book Details & Interaction
1.  **Click on a Book** from the home page.
2.  **Show Details:** Scroll down to show synopsis and existing reviews.
3.  **Add Favorite:** Click the "Heart" icon. Show it turning red.
4.  **Submit Review:**
    - Select a Star Rating (e.g., 5 stars).
    - Type a short review: "Amazing book, highly recommended!"
    - Click "Submit Review".
    - Show the new review appearing immediately in the list.

## 4. Profile Management
1.  **Click "My Profile"** in the nav.
2.  **Show User Info:** Point out the Name and Email.
3.  **Manage Favorites:**
    - Show the book you just favorited in the "My Favorites" list.
    - Click the "X" button to remove it. Verify it disappears.
4.  **Manage Reviews:**
    - Scroll to "My Reviews".
    - Find the review you just wrote.
    - Click "Delete". Verify it disappears.
5.  **Update Profile:**
    - Change the Name (e.g., add " - Updated").
    - Click "Save Changes".
    - Show the success message and the updated name card.

## 5. Conclusion
1.  **Click "Logout".**
2.  **End on Home Page.**
